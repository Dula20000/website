Problem 4 (70 pts)

When designing a second-order control system, we specify how we want it to behave. Two common specifications for step responses are:

  • Overshoot (\(M_p\)): Assuming zero initial conditions, the maximum amount by which the step response exceeds its steady-state value, expressed as a fraction. For example, if the steady-state value is 1 and the response peaks at 1.2, the overshoot is 0.2 (or 20%).
  • Settling time (\(T_s\)): The time after which the response stays within a small band (typically 2%) around the steady-state value.

For an underdamped second-order system (\(0 < \zeta < 1\)) with damping ratio \(\zeta\) and natural frequency \(\omega_n\):

  • Overshoot: \(M_p = e^{-\pi\zeta/\sqrt{1-\zeta^2}}\).
  • Settling time: \(T_s \approx \dfrac{4}{\zeta\omega_n}\).

For an overdamped second-order system (\(\zeta > 1\)):

  • Overshoot: There is no overshoot since the response is not oscillatory.
  • Settling time: Each eigenvalue \(\lambda\) corresponds to a mode \(e^{\lambda t}\). When \(\lambda < 0\), this mode decays, but eigenvalues closer to zero decay more slowly (e.g., \(e^{-0.5t}\) decays slower than \(e^{-2t}\)). The settling time is determined by the slowest-decaying mode: \[T_s \approx \frac{4}{|\lambda|}, \quad \text{where } \lambda \text{ is the eigenvalue closest to zero.}\]

In this question, you will relate design specifications to the eigenvalues of the system and design a closed-loop system that achieves the specifications.

a) Assuming the system is underdamped (\(0 < \zeta < 1\)), find the minimum values of \(\zeta\) and \(\omega_n\) such that the step response settles within \(T_s = 2\) time units and has less than \(M_p = 0.05\) overshoot.

b) Consider a second-order system in state-space form \(\dot{x} = Ax\), where \(x \in \mathbb{R}^2\) and \[ A = \begin{bmatrix} 0 & 1 \\ -\omega_n^2 & -2\zeta\omega_n \end{bmatrix}. \] The eigenvalues of this \(A\) matrix are \[ \lambda = -\zeta\omega_n \pm j\omega_n\sqrt{1-\zeta^2}. \] Using your values of \(\zeta\) and \(\omega_n\) from part (a), compute the eigenvalues that a system would need to meet these specifications.

c) Consider the system:

\[ \dot{x} = Ax + Bu, \qquad A = \begin{bmatrix} 0 & 1 \\ 2 & 0 \end{bmatrix}, \quad B = \begin{bmatrix} 0 \\ 1 \end{bmatrix}. \]

This system is unstable since the eigenvalues of \(A\) are \(\pm\sqrt{2}\). We will stabilize it using state feedback (we will generalize this concept in later lectures). We set \(u = -Kx\), where \(K = \begin{bmatrix} k_1 & k_2 \end{bmatrix}\), which yields \[ \dot{x} = (A - BK)x. \] Compute the matrix \(A - BK\) in terms of \(k_1\) and \(k_2\).

d) Find values of \(k_1\) and \(k_2\) so that the eigenvalues of \(A - BK\) match the eigenvalues you found in part (b).
